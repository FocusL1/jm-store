# 1. Visión del Proyecto

JM-STORE es una plataforma profesional de comercio electrónico desarrollada desde cero con tecnologías modernas.

El proyecto nace con el objetivo de construir una solución escalable, segura y mantenible que permita vender productos físicos y digitales mediante una experiencia de usuario de alta calidad.

JM-STORE no será únicamente una tienda online.

Será una plataforma preparada para crecer y evolucionar durante los próximos años.

---

# 2. Objetivos

Los objetivos principales del proyecto son:

- Construir una arquitectura profesional.
- Mantener un código limpio y organizado.
- Utilizar buenas prácticas de desarrollo.
- Crear una excelente experiencia de usuario.
- Implementar un panel administrativo completo.
- Facilitar el crecimiento futuro del proyecto.

# 4. Filosofía de Desarrollo

JM-STORE se desarrollará siguiendo los siguientes principios:

- Calidad antes que velocidad.
- Arquitectura antes que implementación.
- Componentes reutilizables.
- Código tipado con TypeScript.
- Separación de responsabilidades.
- Documentación continua.
- Seguridad desde el diseño.
- Escalabilidad desde el inicio.

---

# 5. Arquitectura General

La plataforma estará dividida en cuatro grandes bloques.

## Tienda Pública

Todo lo relacionado con los clientes.

## Panel Administrativo

Todo lo relacionado con la administración del negocio.

## Backend

Servicios, autenticación, almacenamiento y base de datos mediante Supabase.

## Documentación

Toda la documentación técnica del proyecto.

# 7. Convenciones del Proyecto

Durante todo el desarrollo seguiremos estas reglas:

- Un componente = una responsabilidad.
- No utilizar `any` salvo casos excepcionales.
- Todo módulo importante tendrá documentación.
- Todo cambio importante quedará registrado en CHANGELOG.
- Todo Sprint actualizará ROADMAP.
- Ninguna funcionalidad importante se desarrollará sin planificación previa.

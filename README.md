## 📖 Descripción

JM-STORE es una plataforma profesional de comercio electrónico desarrollada desde cero utilizando tecnologías modernas.

El objetivo del proyecto es construir una tienda online escalable, segura y de alto rendimiento, siguiendo buenas prácticas de arquitectura de software y documentación.

Este proyecto forma parte de un desarrollo completo que incluye:

- Tienda pública
- Panel administrativo
- Gestión de productos
- Gestión de clientes
- Gestión de pedidos
- Pasarelas de pago
- Dashboard con estadísticas
- Autenticación y control de roles

# ⚙ Instalación

Clonar el proyecto

```bash
git clone https://github.com/TU-USUARIO/jm-store.git
```

Entrar al proyecto

```bash
cd jm-store
```

Instalar dependencias

```bash
npm install
```

Ejecutar

```bash
npm run dev
```

Abrir

```
http://localhost:3000
```

---

# 🔐 Variables de entorno

Crear un archivo:

```
.env.local
```

Agregar:

```env
NEXT_PUBLIC_SUPABASE_URL=

NEXT_PUBLIC_SUPABASE_ANON_KEY=
```
